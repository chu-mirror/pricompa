# マニュアル

大事なことだから先に言っておく、このツールの使用でバンされても自己責任で。

ショートカット説明

+ CTRL+A: 手動用連打
+ CTRL+S: セミオート用連打
+ CTRL+Q: リタイア

二つの連打のキーボード操作は同じ

+ ASDFG: 左から右UB1-UB5
+ スペース: スピード設定
+ C: オートon/off
+ B: 一時停止
+ Q: リタイア

手動用連打は高頻度を保証したが、同時に多数のUBの連打ができない。UBをずっと押して連打を起こす。

セミオート用連打はUBをon/offにする仕様で、多数のUBを同時にonにすることができる。
